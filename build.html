So you did not like binaries. You want to build your own. Don't worry we have your back.

You will need Bash on Windows for this. Run this following command in bash.

wget -qO build https://www.kapadiya.net/jpegoptim-win32/build.sh && sudo bash build

It will build 32bit and 64bit using mxe
