unfortunately Timo Kokkonen does not provide prebuilt windows binaries for jpegoptim.

lucky for you, you can download some not-official-use-on-your-own-risk builds here:

version 1.4.4

windows (32-bit): jpegoptim-1.4.4-win32-binary.zip (173 KB)

windows (64-bit): jpegoptim-1.4.4-win64-binary.zip (185 KB)

how to use (windows)

extract the zip, open cmd.exe and navigate there via cd {folder}

now you can do jpegoptim.exe -m80 -o --force --strip-all --strip-iptc --strip-icc --all-progressive foo.jpg

more information

you can find some instructions on how to use them on the jpegoptim github page.
